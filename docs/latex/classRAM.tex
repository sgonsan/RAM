\doxysubsection{Detailed Description}
Class to represent a \doxylink{classRAM}{RAM}. It contains a 1024 cell memory, a 32 cell register bank and a program counter. The machine can load a program from a file, execute it and print the memory state.~\newline
~\newline
The machine has the following instructions\+: ~\newline
. 


\begin{DoxyItemize}
\item \doxylink{classHalt}{Halt}\+: Stop the execution of the program ~\newline

\item \doxylink{classAdd}{Add}\+: Sum two registers and store the result in a register ~\newline

\item \doxylink{classSub}{Sub}\+: Subtract two registers and store the result in a register ~\newline

\item \doxylink{classMul}{Mul}\+: Multiply two registers and store the result in a register ~\newline

\item \doxylink{classDiv}{Div}\+: Divide two registers and store the result in a register ~\newline

\item \doxylink{classLoad}{Load}\+: Put a value from memory to a register ~\newline

\item \doxylink{classStore}{Store}\+: Put a value from a register to memory ~\newline

\item \doxylink{classRead}{Read}\+: Put a value from the input to a register ~\newline

\item \doxylink{classWrite}{Write}\+: Put a value from a register to the output ~\newline

\item \doxylink{classJump}{Jump}\+: Make a jump to a label ~\newline

\item \doxylink{classJzero}{Jzero}\+: Make a jump if the register is zero ~\newline

\item \doxylink{classJgtz}{Jgtz}\+: Make a jump if the register is greater than zero ~\newline

\item \doxylink{classJltz}{Jltz}\+: Make a jump if the register is less than zero ~\newline
~\newline
The machine can also check if the memory is all zeros and print the memory state. 
\end{DoxyItemize}
